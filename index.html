EGRET

R package for analysis of long-term water quality and streamflow

Download this git project as a .zip file Download R source tar.gz file

Introduction

The EGRET package was created for the analysis of long-term changes in water quality and streamflow.

Download and Installation Options:

- New to R?

If you are new to R, you will need to first install the latest version of R, which can be found here. Although there are many options for running and editing R code, a nice environment to learn R is RStudio. RStudio can be downloaded here. Once you have set up your R environment, proceed to the next section to install the dataRetrieval and EGRET packages.

- R User?

To install the package, first install the required packages from CRAN:


install.packages(c("zoo", "fields", "survival", "spam","plyr"))

Then use the following command:


install.packages("dataRetrieval", repos="http://usgs-r.github.com/EGRET/", type="source")
install.packages("EGRET", repos="http://usgs-r.github.com/EGRET/", type="source")

Further instructions on installation can be found here.

- R Developer?

If you are an experienced R developer, and have the package 'devtools' and all the required LaTex tools installed to create packages, you can build the working version of this package directly from gitHub. Windows tools can be found here. Carefully follow the directions, and note that you need to install Rtools from that page, as well as MikTex (the Windows set of LaTex tools). The full set of Mac LaTex tools can be found here.

Once that is installed, you can use the function install_github to install this package directly from github: 

library('devtools')
install_github("dataRetrieval", "USGS-R")
This will install the latest code that is currently committed to gitHub.
